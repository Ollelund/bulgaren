Bulgaren PWA

Filer:
- index.html
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

Publicera mappen på en HTTPS-webbserver, t.ex. GitHub Pages, Netlify eller Vercel.
Öppna sedan sidan i Chrome på Android och välj "Installera app".

Funktioner:
- Ange maxvikt
- 8-setsschemat för Bulgaren
- Arbetsvikter avrundas till närmaste 2,5 kg
- Grön/röd markering för klarat/misslyckat
- Automatisk 4-minuters vilotimer
- Ett pip vid 30 sekunder kvar
- Tre pip när vilan är slut
- +2,5 kg maxvikt efter 8/8 godkända set
- Samma maxvikt om något set misslyckas
- Lokal historik i webbläsaren
- Offline-stöd via service worker
